# Lista-Compra



Este es un proyecto destinado a crear una aplicación que permita crear y gestionar una lista de la compra. 



El programa de la aplicación constará de un menú de opciones que ofrecerá diferentes funcionalidades para gestionar una lista de la compra. Éstas comprenden añadir un producto a la lista, eliminar un producto, buscar un producto en la lista,

mostrar la lista, vaciar lista y por último salir.



Al iniciar el programa, se pedirá que se introduzca un número para seleccionar una de las opciones mostradas. Esto permitirá ejecutar la funcionalidad deseada, como añadir un producto, por ejemplo. Una vez que se haya ejecutado una funcionalidad, el programa pedirá introducir de nuevo un número correspondiente a una opción. 

Si se desea ejecutar otra función, se introducirá el número correspondiente, mientras que si ya se ha terminado y se desea salir, se introducirá el número de opción que corresponda. Una vez que se haya salido, si se quiere acceder de nuevo al programa, habrá que iniciarlo otra vez, momento a partir del cual se repetirá el mismo proceso.
